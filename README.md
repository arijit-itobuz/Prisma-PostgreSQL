# Prisma + PostgreSQL

## Dependencies

- typescript
- ts-node
- @types/node
- prisma
- express
- @types/express

## Install

- yarn add typescript ts-node @types/node prisma express @types/express

## Commands

- yarn prisma init
- npx tsc --init
- yarn prisma db push
- npx prisma migrate dev --name <migration_name>
- npx prisma studio

## Server

- yarn dev
